# Imagery

This document describes the visual style used throughout KryoNex.

## Visual Style

Imagery should communicate:

- Innovation
- Engineering
- Digital transformation
- Enterprise technology
- Automation
- Reliability

## Preferred Subjects

Examples include:

- Enterprise software
- Artificial Intelligence
- Cloud infrastructure
- Industrial IoT
- Digital engineering
- Data visualization
- Modern workplaces
- Technology infrastructure

## Design Principles

Images should be:

- High quality
- Professional
- Relevant
- Consistent with the KryoNex brand

Avoid generic or low-quality imagery.

## Accessibility

Images should support accessibility by using descriptive alternative text where appropriate.
