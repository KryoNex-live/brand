# Logo Usage

This document defines the correct usage of the official KryoNex logo.

## Official Assets

The official logo assets are located in the `logos/` directory.

- `kryonex-logo.png`
- `kryonex-symbol.png`

Always use the latest version from this repository.

---

# Logo Types

## Company Logo

The primary KryoNex logo should be used for:

- Website headers
- Documentation
- GitHub repositories
- Presentations
- Marketing materials
- Business proposals

---

## Brand Symbol

The brand symbol is intended for:

- Application icons
- Repository avatars
- Social media profile images
- Favicons
- Compact branding

---

# Minimum Size

Use a size that keeps the logo clear and legible.

Avoid reducing the logo to the point where text or design elements become difficult to recognize.

---

# Clear Space

Maintain sufficient empty space around the logo.

No text, graphics, or interface elements should overlap or touch the logo.

---

# Backgrounds

The official logo uses a transparent background.

It may be placed on both light and dark backgrounds provided sufficient contrast is maintained.

---

# Do Not

Do not:

- Stretch the logo.
- Distort proportions.
- Rotate the logo.
- Change the colors.
- Apply filters or effects.
- Add shadows or outlines.
- Crop the logo.
- Recreate the logo manually.

---
