# Favicons

This directory contains the official favicon and platform icon assets used by KryoNex websites.

## Included Assets

- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png
- android-chrome-192x192.png
- ms-icon-150x150.png
- ms-icon-310x310.png
- manifest.json

## Purpose

These assets provide platform-specific icons for:

- Web browsers
- Mobile devices
- Progressive Web Apps (PWA)
- Microsoft tiles

## Usage

These files should be deployed together when publishing a KryoNex website.
