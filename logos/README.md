# Logos

This directory contains the official KryoNex logo assets.

## Assets

### kryonex-logo.png

Primary company logo.

Recommended for:

- Website headers
- Documentation
- GitHub repositories
- Presentations
- Marketing materials

---

### kryonex-symbol.png

Official brand symbol.

Recommended for:

- Favicons
- Application icons
- Social profile images
- Repository avatars
- Small-space branding

---

## Usage

- Maintain the original aspect ratio.
- Do not stretch or distort the logo.
- Do not modify colors.
- Do not add visual effects.
- Do not crop the logo.
- Always use the original asset from this repository.

## Backgrounds

The logo assets use a transparent background and are suitable for both light and dark interfaces.

Always reference the latest files contained in this directory.
